# Node Postgres Boilerplate

A simple boilerplate to work with Node and Postgres

## Features
* Node
* Postgres
* Express
* Sequelize
* Nodemon
* Prettier
* Sucrase

## How to Install

```
yarn
```

## How to use

Development server:
```
yarn dev
```

Debugger:
```
yarn dev:debug
```
